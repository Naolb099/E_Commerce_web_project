Welcome to the Demo E-commerce Website!
This website is built using HTML, CSS, and JavaScript and is intended to serve as a demonstration of a basic e-commerce website.

Features
Product listing page
Product detail page
Shopping cart
Checkout page
Order confirmation page
Getting Started
Clone or download the repository from GitHub
Open the index.html file in your browser to view the website
Browse through the products and add them to your cart
Proceed to checkout and fill in your details
Confirm your order
